# Database Administrator Market Expectations

Database administrator resumes should show database reliability, backup and recovery, performance tuning, security, migration, monitoring, and operational discipline.

Common signals include SQL Server, PostgreSQL, MySQL, Oracle, MongoDB, backup, restore, replication, high availability, indexing, query tuning, stored procedures, database migration, access control, encryption, monitoring, disaster recovery, and capacity planning.

Hiring teams expect evidence that the candidate protects data, improves performance, and supports stable production systems.

Good DBA bullets:

- Tuned slow SQL queries using indexes, execution plans, and schema adjustments, reducing query response time.
- Managed backup and restore processes with regular validation to support disaster recovery readiness.
- Configured database users, roles, permissions, and audit controls to improve access security.
- Supported database migration with validation checks, rollback planning, and minimal downtime.

Weak DBA bullets only list database names. Strong bullets show reliability, performance, security, and operational outcomes.
